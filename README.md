# power-consumption-prediction
This project analyzes how weather affects electricity usage across different zones using machine learning.

## Tools
- Python
- Pandas
- Scikit-learn

## Results
- Zone 1 showed weak performance with high error, highlighting limits of simple linear regression
- Zone 2 had the lowest prediction error (RMSE), making it the most accurate model
- Zone 3 had the highest R², capturing overall trends better than the others
